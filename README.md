### 2. Typography
AicroStrategy uses two specific typefaces to distinguish between functional copy and display visuals.

#### Primary Typeface: **Be Vietnam Pro**
* **Source:** Google Fonts
* **Usage:** Body copy, UI labels, subheadings, and standard headlines.
* **Weights:**
    * *Regular:* Default body copy.
    * *Medium:* Subheadings and UI labels.
    * *SemiBold:* Primary subheadings and key figures.
    * *Bold:* Section titles.

#### Display Typeface: **Schmalfette Grotesk**
* **Style:** Bold, condensed grotesque (Akira-inspired).
* **Usage:** Reserved strictly for visual impact in artwork and key visuals. **Uppercase only**.
* **Note:** Do not use for body text or extended reading.

### 3. Logo System
* **Wordmark:** Custom typeface. Clearspace defined by 50% of its height.
* **Profilemark (AI Monogram):** Stylized "AI" used for social icons and dApp interfaces where space is restricted. Clearspace defined by 1/4 of the circle's diameter.
* **Minimum Sizes:**
    * *Digital Wordmark:* 150px
    * *Digital Profilemark:* 75px
